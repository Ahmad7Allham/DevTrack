# DevTrack

A job application tracker built with Next.js, TypeScript and Tailwind CSS.

## Features

- Add job applications
- Delete applications
- Search by company
- Filter by status
- Statistics dashboard
- LocalStorage persistence
- Dark mode
- Responsive design

## Tech Stack

- Next.js
- TypeScript
- Tailwind CSS
- React
